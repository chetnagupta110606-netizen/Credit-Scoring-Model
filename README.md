# 💳 Credit Scoring Model using Machine Learning

A Machine Learning-based **Credit Scoring Prediction System** built using **Python, Scikit-learn, Streamlit, Pandas, and Matplotlib**. This application predicts whether a customer is a **Good Credit Risk** or **Bad Credit Risk** using a Logistic Regression model.

The project features an interactive Streamlit web application where users can upload a credit dataset, preprocess the data, train the model, evaluate its performance, and visualize important insights through graphs and performance metrics.

---

## 📌 Project Overview

Credit scoring is one of the most important applications of Machine Learning in the banking and financial sector. Financial institutions use credit scoring models to determine whether an applicant is likely to repay a loan.

This project implements a complete Machine Learning workflow including:

- Data Upload
- Data Preprocessing
- Data Visualization
- Model Training
- Model Evaluation
- Feature Importance Analysis
- Interactive Web Interface

## 📚 Machine Learning Workflow

1. Upload the dataset.
2. Preview and inspect the dataset.
3. Analyze dataset statistics.
4. Encode categorical features using LabelEncoder.
5. Split the dataset into training and testing sets.
6. Standardize numerical features using StandardScaler.
7. Train a Logistic Regression model.
8. Evaluate model performance.
9. Visualize results using charts and graphs.
